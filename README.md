# MyFutures
나를 위한 선물

선물거래 - 가격이 움직일 것이라는 예측
1포인트움직이는데 얼마나 이득을 볼지 결정해야함 그때 결정하는것이 랏수
나스닥 기준 1랏을 진입하면 1포인트 움직이는데 20달러 손익이 발생하는것

레버리지는 계약규모에 대한 초기증거금을 설정할수있게한다
나스닥의 계약규모는 나스닥 지수 x 20$
나스닥 지수가 20000이라고 한다면 계약규모는 40000$

내가 레버리지를 500:1로 설정했다면
40000 / 500 = 800 
800$만 있으면 1계약을 진입할수있다는것이다.
이때 지수가 100포인트 변한다면 2000$의 변동성을 가지기때문에 청산당할 위험이 있다.
하지만 우리는 증거금의 1% 비율로 진입할것이기때문에 사실상 증거금은 의미가 없다.
(작은 비중으로 엄청 많이 진입하지않는이상)  

NAS100+ 
계약 규모 : 20$ x 나스닥 지수
1랏 : 20$/포인트 
예시 : 나스닥 20000포인트 -> 400000$
1랏 기준 100포인트 변동시 2000$
수수료 : 1랏당 0.7$

XAUUSD+
계약 규모(1랏) : 100온스 x 골드 가격
1랏 : 100온스
예시 : 1온스당 2500$ -> 250,000$
1랏 기준 10달러 변동시  1000$
수수료: 1랏당 15$

USOUSD+
계약 규모(1랏) : 1000배럴 x 오일 가격 
1랏 : 1000배럴 
예시 : 1배럴당 75.00$ -> 75,000$
1랏 기준 0.1$ 변동시 100$
수수료: 1랏당 2$

EURUSD+
계약 규모(1랏) : 100,000유로 x 유로환율
1랏 : 100,000유로
예시: 1유로당 1.10$ -> 110,000$
1랏 기준 0.001 변동시(환율 0.1% 상승) -> 110$
수수료: 1랏당 15$

NAS100+ : 0.1 - 100포인트(100) - 10달러
XAUUSD+ : 0.01 - 10포인트(10) - 10달러
USOUSD+ : 0.5 - 20포인트(0.2)  - 10달러
EURUSD+ : 0.1 - 10포인트(0.001) - 10달러


1. 가격 변동성의 중요성
해외 선물에서는 종목별로 포인트 단위도 다르고 변동성도 다르기때문에
1%의 수익을 내기위한 진입랏수가 다 달라진다
종목의 변동 특성을 반영하여 가장 일반적인 가격 움직임에서 1%를 벌수있는 진입랏수를 계산해보았다.
하지만 이 또한 사람마다 관점이 다르기 때문에 이 움직임이 짧거나 길다고 생각할수있다.
이럴경우 만약 나는 골드에서 5포인트로 10달러를 벌고싶다고하면
한 진입에서 0.02를 진입하면된다.
하지만 어느 상황에서나 가장 중요한것은 1% 목표 진입이다.
하루에 최소 매매 5번 최대 매매 10번
1000달러 기준 1%의 수익을 5번만 만들면 50달러
한달 거래일수 20일동안 차분하게 하루에 50달러를 만들면 1000달러를 벌수가있다.
